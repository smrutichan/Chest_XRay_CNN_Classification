# Project Documentation

This folder contains the documentation related to the **Chest X-ray Classification using CNN** project.

## Contents

- **Chest_Xray_CNN_Project_Report.pdf**  
  A detailed project report explaining the problem statement, dataset preparation, CNN architecture, training methodology, evaluation results, and K-Fold cross validation analysis.

## Purpose

The report provides a complete overview of the project, including both theoretical background and practical implementation details. It is intended for academic evaluation, project review, and reference.

For code implementation and training scripts, please refer to the root directory of the repository.
